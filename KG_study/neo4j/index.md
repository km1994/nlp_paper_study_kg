## 一、动机篇

### 1.1 为什么 要建索引？

对于 数据库 数据成 百万级、千万级时，对数据库的查询将变得十分耗时，这时需要 在数据库中创建索引来提高查询效率。


## 二、介绍篇

### 2.1 如何创建 Neo4j索引？

```s
    CREATE INDEX ON :LABEL_NAME(property)
```

### 2.2 如何删除 Neo4j索引？

```s
    DROP INDEX ON :LABEL_NAME(property)
```

## 三、缺点篇

### 3.1 索引有什么问题?

会在数据库中创建冗余的副本，因此会占用一些硬盘资源，从而到使得写入速度变慢。
